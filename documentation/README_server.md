<h2>Server</h2>

### Server Info

#### what can this server do?
- the user can get data of:
    - domes
    - users
    - companies
    - oxygen leaks
    - population
    - appointments
    - medical dispatches
    - meteor showers
    - dust storms
- the user can also make a appointment and delete a appointment
